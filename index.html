🚀 What is an Array?

So far, every variable you've declared holds one value at a time. But what if a Spotify playlist has 50 songs? Or a WhatsApp group has 100 members? You can't write int member1, member2, member3... for 100 people! That's where arrays come in.

📱 The Real-World Analogy: A WhatsApp Group Chat

Think of a WhatsApp group called "CFBC CS Class". It's a single group (one name), but it contains many members. An array works exactly the same way — one name, many values stored in contiguous memory slots.

The Official Definition

An array is a container in which numerous variables of the same data type are stored in contiguous (side-by-side) memory locations. Array elements are accessed by their index numbers.

Two Types of Arrays in C

📦 One-Dimensional (1D)

A single list of values — like a Spotify playlist, a leaderboard, or a row of seats.

int scores[5] = {95, 88, 72, 90, 61};

🗂 Multi-Dimensional (2D)

A grid of values — like a Minecraft world map, a seating plan, or a spreadsheet.

int grid[3][3] = {{1,2,3},{4,5,6},{7,8,9}};

🔢 Zero-Based Indexing — The Most Important Rule!

In C, arrays start counting at index 0, not 1. This confuses almost every beginner at first, but once it clicks it never leaves you.

⚠️ If you declare int arr[5]; — the valid indices are 0, 1, 2, 3, 4

There is NO arr[5]! That would be index 5 on a 5-element array — it's out of bounds and causes nasty bugs. The last valid index is always size - 1.

📦 One-Dimensional Arrays

A 1D array is a linear (straight-line) list of values of the same type. Think Spotify queue, TikTok For You page ordering, or a gaming leaderboard.

Syntax: Declaring a 1D Array

data_type  array_name[size];

The size must be an integer constant greater than zero — it tells C how many memory slots to reserve.

Three Ways to Create Arrays

MethodCodeWhat happens
Declare onlyint scores[5];5 slots reserved, values are garbage (undefined)
Declare + full initialiseint scores[5] = {90,85,70,95,88};All 5 values set at declaration
Partial initialiseint scores[5] = {90,85};First 2 set, rest automatically become 0

💡 Shortcut: Let C Count the Size!

If you provide values, you can omit the size and C figures it out:

int scores[] = {90, 85, 70, 95, 88};  // C knows size = 5

Reading & Writing Individual Elements

#include <stdio.h>
int main() {
    int playlist[5];    // declare 5-element array

    // Assign individual elements
    playlist[0] = 6;    // first element
    playlist[1] = 7;    // second element — wait... 6 and 7? 🤔
    playlist[2] = 120;
    playlist[3] = 240;
    playlist[4] = 360;

    // Access individual elements
    printf("First BPM: %d\n", playlist[0]);  // prints 6
    printf("Second BPM: %d\n", playlist[1]); // prints 7
    printf("Last BPM: %d\n", playlist[4]);  // prints 360

    return 0;
}

🔁 Traversing Arrays with Loops

Accessing array elements one-by-one manually (like we did in Lab 1) is fine for small arrays. But what if you have 1000 songs? You need a loop to traverse (go through) the array automatically!

🎮 The Magic Connection: Loops + Arrays = Power Combo

A loop counter i that goes from 0 to size-1 maps perfectly to array indices! This is why arrays and for-loops are basically best friends in C.

Pattern: Read Array from User Input

#include <stdio.h>
int main() {
    int arr[10];
    int i;

    printf("Enter elements of array: ");
    for (i = 0; i < 10; i++) {           // i goes 0,1,2...9
        scanf("%d\n", &arr[i]);           // &arr[i] = address of slot i
    }

    printf("\nElements of array are:\n");
    for (i = 0; i < 10; i++) {           // second pass to print
        printf("%d\n", arr[i]);
    }
    return 0;
}

👁 How the Loop Accesses Each Element

When i = 0: reads/writes arr[0]
When i = 1: reads/writes arr[1]
When i = 9: reads/writes arr[9] ← last valid index
When i = 10: loop exits (condition i < 10 is now false)

#include <stdio.h>
int main() {
    int sales[7] = {120, 145, 200, 98, 310, 275, 180};

    max = sales[0];   // assume first element is max
    min = sales[0];   // assume first element is min

        if (sales[i] < min) min = sales[i];  // update min

    printf("Total tickets sold: %d\n", total);
    printf("Best day: %d tickets\n", max);
    printf("Slowest day: %d tickets\n", min);
    printf("Average per day: %.1f\n", (float)total / 7);
    return 0;
}

Your squad of 5 just finished a Fortnite tournament. Accept each player's XP from the user using a loop, then find and display the squad total XP and the player with the highest XP (their index).

  • Declare int xp[5];
  • Use a for loop with scanf to accept 5 XP values

Input loop: for (i = 0; i < 5; i++) { printf("Player %d XP: ", i+1); scanf("%d", &xp[i]); } — notice i+1 in the prompt so players see "Player 1", not "Player 0"!

TikTok Views Reversal

Accept 6 TikTok view counts into an array, then print them in REVERSE order using a loop counting backwards!

  • Declare and fill int views[6]; using a loop
  • Print the array forward first (as normal)
  • Print the array in reverse — use a loop counting DOWN: i = 5; i >= 0; i--
A countdown loop: for (i = 5; i >= 0; i--). Start at the last valid index (5 for a size-6 array), count down to 0. On each step, views[i] gives you the element from the end backwards! 
#include <stdio.h>
int main() {
    int views[6];
    int i;

    for (i = 0; i < 6; i++) {
        printf("Enter view count %d: ", i + 1);
        scanf("%d", &views[i]);
    }

    printf("\n▶️  Forward: ");
    for (i = 0; i < 6; i++)
        printf("%d  ", views[i]);

    printf("\n⏪ Reversed: ");
    for (i = 5; i >= 0; i--)     // start at last index!
        printf("%d  ", views[i]);

    printf("\n");
    return 0;
}

To access element at row r, column c: use array_name[r][c]

Initialising a 2D Array

// Full initialisation with row grouping
int a[3][4] = {
    {0, 1, 2, 3},    // row 0
    {4, 5, 6, 7},    // row 1
    {8, 9, 10, 11}   // row 2
};

Traversing a 2D Array — Nested Loops!

Rule: One loop for rows, one loop for columns — NESTED together

The outer loop counts rows (r). The inner loop counts columns (c). For every single row, the inner loop goes through ALL columns before moving to the next row.

int arr[3][3], r, c;
for (r = 0; r < 3; r++) {          // outer: rows 0,1,2
    for (c = 0; c < 3; c++) {       // inner: cols 0,1,2 for EACH row
        printf("%d\t", arr[r][c]);  // \t = tab for alignment
    }
    printf("\n");                   // newline after each full row
}

//Program: Matrix Addition
#include <stdio.h>
int main() {
    int m, n, c, d;
    int first[10][10], second[10][10], sum[10][10];

    printf("Enter the order of the matrices:\n");
    scanf("%d%d", &m, &n);

    printf("\nEnter elements of first matrix:\n");
    for (c = 0; c < m; c++)
        for (d = 0; d < n; d++)
            scanf("%d", &first[c][d]);

    printf("Enter elements of second matrix:\n");
    for (c = 0; c < m; c++)
        for (d = 0; d < n; d++)
            scanf("%d", &second[c][d]);

    for (c = 0; c < m; c++)
        for (d = 0; d < n; d++)
            sum[c][d] = first[c][d] + second[c][d];

    printf("\nSum of entered matrices:\n");
    for (c = 0; c < m; c++) {
        for (d = 0; d < n; d++)
            printf("%d\t", sum[c][d]);
        printf("\n");
    }
    return 0;
}

Matrix Multiplication — The Boss Level

This is the challenge from your textbook! Accept the dimensions of two 2D arrays. Check if multiplication is valid (columns of first = rows of second). If valid, perform matrix multiplication.

🧮 Matrix Multiplication Rule

For multiply[c][d]: sum ALL the products first[c][k] * second[k][d] for k = 0 to (n-1). This requires THREE nested loops: row of first, column of second, and the inner product loop.

Result: multiply[c][d] += first[c][k] * second[k][d]

  • Accept m×n dimensions for first matrix
  • Accept p×q dimensions for second matrix
  • Check: if n ≠ p, print "Cannot multiply!" and exit
  • Otherwise: use THREE nested loops (c, d, k) to compute product
  • Reset sum=0 before each new element computation!
The multiplication loops: for(c=0;c<m;c++) { for(d=0;d<q;d++) { sum=0; for(k=0;k<p;k++) { sum += first[c][k]*second[k][d]; } multiply[c][d]=sum; } }
Why sum = 0 inside the outer two loops? Because each element of the result matrix is a fresh accumulation. If you don't reset, you keep adding from the previous element's calculation! Put sum = 0; right before the innermost loop. 
#include <stdio.h>
int main() {
    int m, n, p, q, c, d, k, sum = 0;
    int first[10][10], second[10][10], multiply[10][10];

    printf("Rows and cols of first matrix: ");
    scanf("%d%d", &m, &n);

    printf("Rows and cols of second matrix: ");
    scanf("%d%d", &p, &q);

    if (n != p) {
        printf("Matrices can't be multiplied! (cols of A must equal rows of B)\n");
        return 1;
    }

    printf("Enter elements of first matrix:\n");
    for (c = 0; c < m; c++)
        for (d = 0; d < n; d++)
            scanf("%d", &first[c][d]);

    printf("Enter elements of second matrix:\n");
    for (c = 0; c < p; c++)
        for (d = 0; d < q; d++)
            scanf("%d", &second[c][d]);

    // Three nested loops: row c, col d, inner product k
    for (c = 0; c < m; c++) {
        for (d = 0; d < q; d++) {
            sum = 0;                        // reset for each new cell!
            for (k = 0; k < p; k++) {
                sum += first[c][k] * second[k][d];
            }
            multiply[c][d] = sum;
        }
    }

    printf("\nProduct matrix:\n");
    for (c = 0; c < m; c++) {
        for (d = 0; d < q; d++)
            printf("%d\t", multiply[c][d]);
        printf("\n");
    }
    return 0;
}

🧪 LAB 7 — THE MAIN EVENT

SKN Soca Festival Scorer (5 Performers, 3 Judges)

  • Declare: int scores[5][3]; and float avg[5];
  • Use nested loops to accept all 15 scores (5 performers × 3 judges)
  • Use nested loops to compute each performer's average (sum the 3 judge scores, divide by 3.0)
  • Find the winner: track which performer index has the highest average
  • Print a full scoreboard table AND announce the winner!
For averages: loop through performers (p=0 to 4). Inside, loop through judges (j=0 to 2) and sum: total += scores[p][j];. After inner loop: avg[p] = (float)total / 3;. Reset total=0 before each performer's inner loop!
For winner: declare int winIdx = 0; before the check loop. Then: if (avg[p] > avg[winIdx]) { winIdx = p; }. At the end, winIdx+1 is the winner's number and avg[winIdx] is their average.
Why (float)total / 3 instead of total / 3? Because integer division truncates! 88+92+79 = 259. 259/3 in int-land = 86 (drops the .3). Casting to float: 259.0/3 = 86.3. Always cast when you need decimals! 
#include <stdio.h>
int main() {
    int   scores[5][3];
    float avg[5];
    int   p, j, total, winIdx = 0;

    // ── Accept all scores ──
    for (p = 0; p < 5; p++) {
        printf("\n--- Performer %d ---\n", p + 1);
        for (j = 0; j < 3; j++) {
            printf("  Judge %d score: ", j + 1);
            scanf("%d", &scores[p][j]);
        }
    }

    // ── Compute averages ──
    for (p = 0; p < 5; p++) {
        total = 0;
        for (j = 0; j < 3; j++)
            total += scores[p][j];
        avg[p] = (float)total / 3;
    }

    // ── Find winner ──
    for (p = 1; p < 5; p++)
        if (avg[p] > avg[winIdx]) winIdx = p;

    // ── Print scoreboard ──
    printf("\n🎺 ====== SKN SOCA FESTIVAL SCORES ====== 🎺\n");
    printf("%-12s %8s %8s %8s %8s\n",
        "Performer", "Judge 1", "Judge 2", "Judge 3", "Average");
    printf("--------------------------------------------------\n");

    for (p = 0; p < 5; p++) {
        printf("Performer %-2d %8d %8d %8d %8.1f",
            p + 1, scores[p][0], scores[p][1], scores[p][2], avg[p]);
        if (p == winIdx) printf(" 🏆 WINNER!");
        printf("\n");
    }
    printf("\n🎉 Congratulations Performer %d! Average: %.1f\n",
        winIdx + 1, avg[winIdx]);
    return 0;
}
